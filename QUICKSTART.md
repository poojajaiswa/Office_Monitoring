## Installation (3 minutes)

1. **Navigate to project directory**:
```bash
cd Office-Monitoring-with-ai-yolov8-master
```

2. **Install dependencies**:
```bash
pip install -r requirements.txt
```

## Running the App (1 minute)

```bash
streamlit run app.py
```

That's it! Your browser will open automatically at `http://localhost:8501`

## First Time Use

1. **Click "Start Processing"** (uses sample video by default)
2. **Wait** for processing to complete (~1-2 minutes)
3. **View** the processed video with annotations
4. **Download** your results

## Upload Your Own Video

1. Select **"Upload Custom Video"** in the sidebar
2. Click **"Browse files"** and choose your video
3. Click **"Start Processing"**
4. Download the processed result
